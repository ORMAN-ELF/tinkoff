Назначение приложения: Данное приложение предназначено для сбора данных о пользователях и генерации на основе этих данных файла формата-excel.
Систем сборки в проекте maven.
Основной файл - "ExcelFile.java".

Следуйте следующим пунктам чтобы запустить проект:
1. Открыть консоль
2. Сменить текущую директорию на директорию проекта
3. Написать в консоли: mvn compile
4. Написать в консоли: mvn exec:java -Dexec.mainClass="com.app.ExcelFile"
5. Готово. В директории src/main/ появился файл Excel
