Домашнее задание №2

В папке "java" (путь до папки "/src/main/") находятся 2 файла.

Основной файл "ExcelFile.java" предназначен для создания файла Excel со собранными данными о пользователях.

Файл "DataModel.java" предназначен для сбора данных о пользователях.
Данные о пользователе: 
1. Имя. 
2. Фамилия
3. Отчество
4. Возраст
5. Пол
6. Дата рождения
7. ИНН
8. Почтовый индекс
9. Страна
10. Область
11. Город
12. Улица
13. Дом
14. Квартира

В папке "resources" (путь до папки "/src/main/") находятся файлы txt формата с данными для полей "Имя", "Фамилия", "Отчество", "Страна", "Город", "Улица"
